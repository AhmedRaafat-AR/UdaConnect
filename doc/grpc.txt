open modules/location/

to make a sample request

1- run 'sh script/create_kafka_topic.sh' to create kafka topic

2- run 'sh script/consumer.sh' that will make a request to kafka and write the location data to msg file and 'cat msg'.

2- run 'sh script/req_grpc.sh' that will make a request to grpcserver and pass the location data to kafka producer.

4- kafka producer will make a request to kafka server with 'locations' topic to add message.


-Portobuf location message structured data 

message LocationMessage {
  int32 userId = 1;
  int32 latitude = 2;
  int32 longitude = 3;
}

service locationService {
  rpc Create(LocationMessage) returns (LocationMessage);
}
