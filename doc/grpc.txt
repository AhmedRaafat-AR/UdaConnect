grpc documentation of location producer, open modules/location/writer.py to know how to make a sample request.

writer.py will send request to grpc server then pass the data to location producer.

structured data 

message LocationMessage {
  int32 userId = 1;
  int32 latitude = 2;
  int32 longitude = 3;
}

service locationService {
  rpc Create(LocationMessage) returns (LocationMessage);
}
